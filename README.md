# imba-capacitor-hn

> In progress

Install dependencies
```sh
npm install
```

Add mobile platform (optional)
```sh
npx cap add android
npx cap add ios
```

Web dev
```sh
npm start
```

Build dev/prod versions
```sh
npm run build:dev
npm run build:prod
```

Build prod and open Android/iOS

```sh
npm run build:android
npm run build:ios
```
